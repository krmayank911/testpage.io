Burning bootloader onto a raw ATmega328P can be tricky sometimes depending upon the version you buy from the market. In the market there are two variants available ATmega328-PU and ATmega328P-PU you can find it on your IC just take a closer look.


First of all what is a bootloader?
It is basically a program that resides in a microcontroller chip that manages the upload of your sketches onto the IC. It is basically the operating system of the IC.

Here what you need to do if you have a ATmega328-PU:

  • First find the avrdude.conf in C:\ProgramFiles\arduino\hardware\tools\ avrdude.conf.
  • Copy this file at a safe place(as it will be used for bootloading ATmega328P-PU).
  • Open this file in wordpad and find the signature: 0x1e 0x95 0x0F (for ATmega328P-PU) and change it with 0x1e 0x95 0x14 (for ATmega328-PU).
  • Save it and use it for bootloading the ATmega32-PU.
Note: If you want to bootload simply ATmega328P-PU, there is no need to change the above file and you can directly proceed for the connections.

Connections of ATmega328 IC with Arduino Uno:

  • UNO 5V —> ATmega328 pin 7 (VCC)
  • UNO GND —> ATmega328 pin 8 (GND)
  • UNO pin 10 —> ATmega328 pin 1 (RESET)
  • UNO pin 11 —> ATmega328 pin 17 (MOSI)
  • UNO pin 12 —> ATmega328 pin 18 (MISO)
  • UNO pin 13 —> ATmega328 pin 19 (SCK)

Additional Components

  • A 16 MHz crystal
  • A 10k resistor, and
  • Two 18 to 22 picofarad (ceramic) capacitors.

Note: additional circuitery can be avoided, to know more visit Arduino.cc

Circuit:

In the Arduino IDE:

  • Upload the Arduino ISP program from the Example onto the Uno board.
  • Exchange the files.
  • Select the Correct Board and Serial Port from the tool menu.
  • Select the programmer ->Arduino as ISP in the tools menu.
  • Click on the burn Bootloader in the tools menu.

And thats it. Your IC is now bootloaded.
